Pancake Themer Customize your pancake design

This tool helps make basic style adjustments to Pancake's default theme. It hasn't been tested much, so use at your own risk!
Customize your design using the controls below, then click the Get CSS button and copy the code into Pancake's Frontend CSS box.
Some controls affect multiple pages, so be sure to look at each page using the "Current Page" dropdown menu. Hide this header.
